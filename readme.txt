A WordPress plugin that extends the Recipress Plugin.

== Description ==

A WordPress plugin that extends the Recipress Plugin to allow for data in main RSS Feed, cleans output for Recipes with no Instructions, allows for printable Recipes.

= Features =

* Adds Recipes to Main Blog RSS Feed, including Images
* Allows for Recipes to be Printed
* Cleans output on recipes with no instruction or no ingredient
* Option to unlink Ingredients links or Taxonomies links on output
* Works in WordPressMU
* Support for Recipress 1.9.4 and Recipress 1.9.5
* Added Google recipes markup. See https://support.google.com/webmasters/answer/173379 for details

= Note =

This will be the last update, we are not going updates anymore, because Recipress are not upgrading their plugin for a long time.

== Installation ==

1. Download & Unzip and Upload all files to a sub directory in "/wp-content/plugins/".
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Go to Options under Settings for Print and Link Options.

== Frequently Asked Questions ==

= Do I need a version of Recipress Installed? =
Yes, you need the latest version of Recipress.
